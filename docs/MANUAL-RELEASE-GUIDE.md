# 📦 מדריך העלאת גרסה חדשה ידנית

## 🎯 תהליך מהיר (5 דקות)

### שלב 1: בניית ה-EXE
```bash
.\build-exe.bat
```

המתן כ-5 דקות עד שהבנייה תסתיים.

---

### שלב 2: מצא את הקובץ
הקובץ נמצא ב:
```
dist-electron\gemach-management-system-setup-2.9.XX.exe
```

---

### שלב 3: העלאה למאגר הציבורי

1. **פתח את GitHub:**
   https://github.com/sh5616107/gemach-releases/releases

2. **לחץ "Create a new release"** (כפתור ירוק)

3. **מלא את הפרטים:**
   - **Choose a tag:** `v2.9.XX` (הגרסה החדשה)
   - **Release title:** `מערכת ניהול גמ"ח v2.9.XX`
   - **Description:** (העתק מלמטה)

4. **העלה את הקובץ:**
   - גרור את קובץ ה-EXE לאזור "Attach binaries"
   - או לחץ "choose your files" ובחר את הקובץ

5. **לחץ "Publish release"** (כפתור ירוק למטה)

---

### שלב 4: עדכן את המאגר הפרטי

```bash
git add .
git commit -m "Release v2.9.XX"
git push
git tag v2.9.XX
git push origin v2.9.XX
```
