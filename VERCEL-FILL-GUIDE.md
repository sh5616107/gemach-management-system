# 📝 מה למלא ב-Environment Variables - מדריך מדויק

## התמונה שראית

אתה רואה טופס עם:
- **Key** (משמאל) - שם המשתנה
- **Value** (מימין) - הערך של המשתנה
- כפתור **"Add More"** להוספת משתנה נוסף

---

## 🎯 מה בדיוק למלא

### משתנה ראשון:

**בשדה Key (משמאל):**
```
VITE_USE_API
```

**בשדה Value (מימין):**
```
true
```

לחץ על **"Add More"** (+ Add More)

---

### משתנה שני:

**בשדה Key (משמאל):**
```
VITE_API_URL
```

**בשדה Value (מימין):**
```
https://gemach-management-system-production.up.railway.app
```

## ⚠️ חשוב מאוד!

### אחרי שמילאת את שני המשתנים:

1. **גלול למטה** בדף
2. תראה אפשרות לבחור **Environment** (סביבה)
3. **וודא שמסומן רק "Production"** ✅
4. **אל תסמן** "Preview" או "Development" ❌

זה אמור להיראות ככה:
```
☐ Development
☐ Preview  
☑ Production  ← רק את זה!
```

---

## 🚀 אחרי שמילאת

1. גלול עד הסוף של הדף
2. לחץ על הכפתור הכחול **"Deploy"**
3. חכה 2-3 דקות
4. זהו! 🎉
